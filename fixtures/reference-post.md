# Reference Post Rebuild Notes

Use these notes instead of the original social post.

## Core Flow

1. The page opens on a full-bleed hero with one dominant interactive object in the center.
2. The object reacts to hover or pointer movement with soft elastic motion.
3. A compact control tray appears beneath it with 3 to 4 actions.
4. Triggering an action updates the center object immediately and animates the surrounding labels.

## Visual Direction

- Clean editorial layout with lots of empty space.
- Large display typography.
- Off-white background with charcoal text and one bright accent color.
- Motion should feel springy and precise, not floaty.

## Implementation Notes

- Match the flow and composition, not the exact assets.
- Create placeholder assets locally inside the eval folder.
- Keep the interaction fast and obvious on both desktop and mobile.
